INTRODUCTION

Welcome to the FivGo Privacy Policy.

FivGo is an online-based transportation service application developed as part of an academic project for learning, research, and software development purposes. This application is designed to simulate an online transportation service by providing trip booking features, customer-driver matching, real-time location tracking, trip history management, and other supporting features. In the process of using the application, FivGo may collect and manage certain information necessary to support system operations and user experience. Therefore, FivGo is committed to maintaining the security and confidentiality of user data in accordance with reasonable and responsible data management practices.

This Privacy Policy explains how FivGo collects, uses, stores, discloses, and protects information obtained from users during the use of the application.

By registering, accessing, or using the FivGo application, you acknowledge that you have read, understood, and agreed to the terms described in this Privacy Policy.

By using this application, you agree to the collection and use of data in accordance with this policy.

INFORMATION COLLECTED

FivGo only collects data necessary to operate the FivGo online transportation service.

Personal Information

FivGo may collect personal information such as:

  • Full name
  • Phone number
  • Email
  • Google account information used for authentication

Purpose:

  • Creating and managing accounts
  • User authentication (OTP / Google Login)
  • User identification within the system

Location Data

FivGo may collect location data such as:

  • Real-time device location (precise location)
  • Pickup location
  • Destination location
  • Movement during the trip

Purpose:

  • Finding the nearest driver
  • Displaying trip routes
  • Calculating estimated distance and fare
  • Providing trip tracking features
  • Running FivGo transportation service

Trip Data

FivGo may collect trip data such as:

  • Trip booking details
  • Trip history
  • Trip status (pending, accepted, on-going, completed, cancelled)
  • Trip fare
  • Trip duration

Purpose:

  • Providing transportation services
  • Storing user activity history
  • Handling complaints or disputes

Payment Data

FivGo may collect payment data such as:

  • Payment method (cash / non-cash)
  • Transaction status
  • Transaction history

Purpose:

  • Processing payments
  • Recording trip transactions
  • Verifying payment status

Communication Data

FivGo may collect communication data such as:

  • In-app messages between users and drivers
  • User reports or complaints

Purpose:

  • Facilitating communication between users and drivers
  • Handling service complaints and disputes
  • Supporting service operations

Device and Identification Data

FivGo may collect device data such as:

  • Device ID
  • IP Address
  • Device information (model, OS)
  • Application logs
  • Crash and error data

Purpose:

  • Maintaining system security
  • Bug analysis and fixing
  • Application performance monitoring
  • Preventing misuse

In-App Activity

FivGo may collect in-app activity such as:

  • Booking history
  • Application usage activity
  • Transaction history

Purpose:

  • Providing trip history features
  • Application usage analysis
  • Service improvement

Camera and Gallery Access

The FivGo Customer application may request camera access to allow users or drivers to upload and take photos required for service operational purposes.

Data Usage

FivGo uses the collected data to:

  • Provide FivGo transportation services
  • Connect users with drivers
  • Display trip location and routes
  • Process payments
  • Send service-related notifications
  • Handle reports and disputes
  • Improve application performance
  • Maintain system security

Location Services

Location access is an essential part of the FivGo service. We use location data to determine pickup and destination locations, display real-time trip routes, calculate estimated distance and fare, and provide trip tracking features for user safety and convenience.

Note: Location information is only collected when necessary to provide transportation services and operational monitoring.

Sign In with Google

FivGo supports Google Sign-In authentication to simplify the user registration and login process.
When using this feature, we only receive basic information necessary to create an account, such as the username, email address, and Google account identifier. We do not store or access any other information from your Google account beyond what is required for authentication and identification within the FivGo application.

Payment Processing

FivGo is integrated with DompetX to process non-cash payments. When making a payment, we only send the information necessary to process the transaction, such as the payment amount, available payment methods such as Virtual Account and QRIS, and transaction status.

Note: FivGo does not store or directly handle credit card numbers or other sensitive payment information. All transactions are processed through a secure and trusted third-party provider.

Information Sharing

FivGo shares certain information only to operate the service.

Driver Partners

FivGo shares:

  • Phone number
  • Pickup location

Purpose:

Third-Party Service Providers

We may use third parties such as:

Purpose:

  • Supporting application operations
  • Providing map and tracking features
  • Maintaining system stability

Legal Compliance

We retain data as long as:

  • The account remains active

Data will be deleted or anonymized when no longer needed.
